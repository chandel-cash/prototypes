Afterpay Media Network Opt-Out Prototype

This prototype demonstrates the user flow for opting out of the Afterpay Media Network (AMN) as requested by the Block Privacy Counsel. The prototype follows Square's design style and implements the requirements outlined in the Google Slides document.

How to Use This Prototype

To experience the complete user flow, start with the simulated Square homepage and follow these steps:

1
Open square-homepage.html in your browser
↓
2
Scroll to the footer and click either "Your Privacy Choices" or "Do Not Share My Personal Information" link
↓
3
On the opt-out landing page, click the "Customer Opt-Out" button
↓
4
Fill out the form and click "Submit Opt-Out Request"
↓
5
View the confirmation page
Start Prototype View Option 1 (Do Not Share) View Option 2 (Your Privacy Choices)

Files Included

  • square-homepage.html - Simulated Square homepage with footer links
  • amn-opt-out-prototype.html - Main landing page for the opt-out process (Option 1)
  • amn-opt-out-prototype-alt.html - Alternative design with separate links (Option 2)
  • customer-opt-out-form.html - Form for customers to submit their opt-out request
  • opt-out-confirmation.html - Confirmation page shown after form submission
  • square-logo.svg - Square logo for the prototype
  • prototype-script.js - JavaScript to connect the prototype pages
  • index.html - This instructions page

Implementation Notes

This is a static HTML/CSS prototype. In a real implementation, the forms would submit data to a server and the design would be integrated with Square's existing website infrastructure.

The prototype demonstrates both design options from the slides:

  1. Option 1: Single "Do Not Share My Personal Information" link
  2. Option 2: Separate "Cookie Settings" and "Your Privacy Choices" links
